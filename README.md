# gheart-target-ui

A minimal Todo app used as a target repo for [gheart](https://github.com/ajingu/gheart) (Tinder-style swiping for PR reviews).

## Run

Requires Node.js 18 or newer. This app has no external npm dependencies, so `npm install` is not required.

```bash
npm run dev
```

Then open http://127.0.0.1:3100.

To use another port:

```bash
PORT=3101 npm run dev
```
